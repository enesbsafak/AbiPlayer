# Abi Player (v0.1 Beta)

Electron + React + TypeScript ile gelistirilmis beta asamasinda bir IPTV oynatici.

## Ozellikler

- Xtream Codes API destegi (Canli TV, VOD, Dizi)
- M3U/M3U8 liste ice aktarma (URL veya dosya)
- MPV native oynatma + HTML5/HLS fallback
- EPG (XMLTV) destegi
- Coklu ses ve altyazi secimi
- Harici altyazi yukleme (SRT/VTT/ASS/SSA)
- Film ve dizi detay zenginlestirme (TMDB)
- Favoriler, arama, mini player ve klavye kisayollari

## Kurulum

```bash
git clone https://github.com/enesbsafak/freeiptv.git
cd freeiptv
npm install
```

## Gelistirme

```bash
npx electron-vite dev
```

## Build

```bash
npx electron-vite build
```
